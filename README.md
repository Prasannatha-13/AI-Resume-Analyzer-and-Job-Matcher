
# 🧠 AI-Powered Resume Analyzer & Job Matcher

## 📌 Overview

The **AI-Powered Resume Analyzer & Job Matcher** is a full-stack web application built using the MERN stack that helps users evaluate and improve their resumes. The system analyzes uploaded resumes, extracts key skills, calculates a resume score, and recommends suitable job roles based on the user's profile.

This project simulates a simplified **ATS (Applicant Tracking System)** used by recruiters, providing users with actionable insights to enhance their chances of getting shortlisted.

---

## 🚀 Features

### 🔐 Authentication

* Secure user registration and login
* Password hashing using bcrypt
* JWT-based authentication and protected routes

### 📄 Resume Upload & Parsing

* Upload resumes in PDF format
* Extract text using PDF parsing libraries
* Store and manage uploaded resumes

### 🧠 Resume Analysis

* Identify key skills from resume content
* Analyze education and relevant experience
* Match extracted skills with predefined skill sets

### 📊 Resume Scoring System

* Generate a resume score based on:

  * Skills presence
  * Keyword matching
  * Content quality
* Provide a clear evaluation of resume strength

### 💼 Job Role Recommendation

* Suggest relevant job roles based on extracted skills
* Map skills to industry-standard roles (e.g., Frontend Developer, Backend Developer)

### 📈 User Dashboard

* Display:

  * Resume score
  * Extracted skills
  * Missing skills
  * Recommended job roles
* Simple and intuitive UI for better user experience
